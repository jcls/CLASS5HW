This directory contains homework from the October 2, 2014 BEWD class.
greeting.rb = simple greeting
EXER1.rb = only says hello if user name is Jack or Jill
EXER2.rb = adds or multiplies a number from 1...N
secretnumber.rb = guess the secret number in 3 tries
sparkle.rb = prints out a string with asterisks
modulus.rb = uses modulus with 3 or 5 to print out BEWD, ROCKS, BEWD ROCKS
smallesttemp.rb = reads a file with weather data and determines which day had the most consistent temperatures, ie. least temperature difference
